# Decision D-008 — Phase 1 scope center

Frozen wording direction:

`Phase 1 defines the boxed component model, the minimum atomic workflow blocks, the role assemblies built from them, and the first working end-to-end delivery chain.`

Why this matters:
- corrects the earlier drift toward treating Phase 1 mainly as a delivery engine
- makes the real v2 IP explicit: boxed components, contracts, assemblies, and flows
- establishes that v2 is a foundation-reset first and an implementation system second
- makes the first working delivery chain a proof of the model, not the whole model itself

Implication:
- `MISSION-STATEMENT.md` scope should be revised around this definition
- later docs should expand this into boxed component model, roles, workflows, contracts, and obligations
