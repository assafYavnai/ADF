# Decision D-009 — Phase 1 scope

Frozen wording for `MISSION-STATEMENT.md`:

## Phase 1 scope
- define the ADF startup model
- define the boxed component model that all core system functions must follow
- define the minimum roles and workflows required for the startup model
- build the minimum set of boxed components required to run that model
- prove the model with a first complete end-to-end delivery chain

Related decision:
- success criteria will be defined in a separate section rather than overloaded into the scope section

Why this wording was accepted:
- keeps the mission scope high level
- separates scope from specification detail and implementation order
- centers v2 on the startup model, boxed components, and proof of the model
