# Decision D-035 - Low Trust Requires Action

Frozen decision:
- low trust requires action, not only observation

What that means:
- trust is not only a metric
- the current direction is:
  - `9/10` or higher is the healthy minimum
  - `8/10` means action is required now
- later exact action rules still need freezing

Why this was accepted:
- fire-and-forget delegation cannot tolerate persistently low trust
- this turns trust into a governing signal instead of passive reporting
- it creates the path toward self-healing and self-improving behavior later
