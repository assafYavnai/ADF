2. Conceptual Root Cause

- The new event-first substrate added a broad mutation surface before freezing the route-critical invariant that only merge-backed closeout may create terminal completion truth. The code enforces that rule in `mark-complete`, but not in the shared event reducer.
- The worker-selection model collapses three distinct sources into two labels. The implementation prefers persisted feature-local state ahead of invoker defaults, but the provenance model only exposes `explicit_override` versus `invoker_inheritance`, so continuity reuse is serialized as if it were current-runtime inheritance.
