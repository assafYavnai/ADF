2. Conceptual Root Cause

- Missing contract: universal durable-write provenance policy. The system enforced strict provenance on memory, decision, governance, and `memory_manage`, but telemetry kept a local normalizer and dispatch fallback that invent actor fields instead of sharing the same write-family contract.
- Missing contract: byte-stable transport policy. The launcher hardening targeted “route runs” but did not enforce “argv reaches the target unchanged,” so Windows shim resolution and explicit shell-backed provider paths still allow shell re-parsing to sit in the middle of the route.
- Missing contract: single-sourced capability modeling. Governance family modeling, injectable-memory policy, and the published MCP manage surface are allowed to drift, so `artifact_ref` exists in schema and context policy without an end-to-end callable route.
