# KPI Review-Cycle Lesson

Date: 2026-04-03
Scope: `assafyavnai/adf/phase1`
Feature stream: `coo-kpi-instrumentation`

## Lesson

- Review-cycle report artifacts must use the exact required heading contract (`1. ...`, `2. ...`, `3. ...` etc.), not Markdown `##` headings.
- When `audit-findings.md` or `review-findings.md` violates that artifact contract, the helper treats the report as not reusable and can block or mis-handle downstream `report-surfaced` transitions.
